# Premium Temperature Converter 🌡️

A modern, beautifully designed temperature converter built with HTML, CSS, and Vanilla JavaScript. It allows for seamless conversion between Celsius, Fahrenheit, and Kelvin in a highly interactive and aesthetically pleasing interface.

## 🌟 Features

- **Multi-Unit Conversion**: Convert between Celsius, Fahrenheit, and Kelvin instantly.
- **Modern UI Edge**: Uses a sleek glassmorphism card design, vibrant gradient textures, and Google's modern "Outfit" typeface.
- **Interactive UX**: 
  - Handy "Swap" button to instantly reverse the "From" and "To" units.
  - Smooth scale and fade animations when retrieving results.
  - Interactive hover and focus states with dynamic drop shadows.
- **Fast & Responsive**: Fully responsive design that stays centered and looks premium on any mobile, tablet, or desktop screen.
- **Validation Handling**: Error indicator handling to block off non-numbers and invalid inputs gracefully.

## 🛠️ Technologies Used

- **HTML5**: Semi-semantic tags to structure the form and results.
- **CSS3**: Uses CSS Variables (Custom Properties), flexbox layouts, backdrop-blur for glassmorphism effects, radial gradients, and fluid transitions.
- **JavaScript (Vanilla)**: Handles the form calculations, DOM manipulation, input validation, and swap logic entirely in the browser.

## 🚀 How to Run the Project

1. Download or clone this repository to your local machine.
2. Navigate to the project folder.
3. Simply double-click the `tas3.html` file to open it in any modern web browser.

*Everything is contained in a single HTML file—no local server, build tools, or installations are required!*
